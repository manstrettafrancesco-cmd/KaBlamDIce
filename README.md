# Ka-Blam D10

Web app/PWA per iPhone con un dado D10, 80 vignette e suoni sintetizzati.

## Pubblicazione con GitHub Pages

1. Crea un nuovo repository GitHub.
2. Carica **tutti i file e le cartelle contenuti qui dentro** nella radice del repository.
3. Apri **Settings → Pages**.
4. In **Build and deployment**, scegli **Deploy from a branch**.
5. Seleziona `main` e cartella `/ (root)`, quindi **Save**.
6. Dopo circa un minuto apri l'indirizzo mostrato da GitHub Pages.

Su iPhone: Safari → Condividi → **Aggiungi alla schermata Home**.

## Immagini

Le immagini sono in `assets/images/1` fino a `assets/images/10`, otto per ogni risultato.
